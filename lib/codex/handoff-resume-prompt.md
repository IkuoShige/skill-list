# 再開プロンプトテンプレート

handoffコマンド実行後、ユーザーに以下の形式で再開プロンプトを提示する。
ユーザーはこれを次のCCセッションの最初のメッセージとしてコピペする。

## テンプレート

```
前セッションの引き継ぎを読んで作業を再開してください。

引き継ぎ文書: {handoff-detail-path}

まず引き継ぎ文書を読み、現状を把握した上で、未完了タスクの優先順位を提示してください。
```

## 変数
- `{handoff-detail-path}`: 詳細引き継ぎファイルの絶対パス（handoff.mdが生成時に決定）

## 表示方法
- コードブロックで囲んで表示する（コピペしやすくする）
- 「次のセッションで以下を最初のメッセージとして送信してください:」と前置きする
